# MotorPH Payroll System Database

This repository contains the SQL database structure and test data for the MotorPH Payroll System.

## Files Included

- `DatabaseStructure.sql`  
  Contains the complete database schema and table relationships.

- `01_user_account_role_test_data.sql`  
  Test data for user accounts, roles, permissions, and employees.

- `02_payroll_admin_role_test_data.sql`  
  Test data for payroll, earnings, deductions, and tax brackets.

- `03_time_attendance_test_data.sql`  
  Test data for attendance and overtime tracking.

- `04_leave_management_test_data.sql`  
  Test data for leave types and leave requests.

- `05_salary_calculation_test_data.sql`  
  Test data for departments, positions, salaries, and allowances.

- `06_relationship_verification_queries.sql`  
  SQL queries used to verify table relationships and inserted test data.

## Database Used

- MySQL Workbench
- Database Name: `payrollsystem_db`
